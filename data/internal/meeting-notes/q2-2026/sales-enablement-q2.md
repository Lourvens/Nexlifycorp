## 5. Security Confidence Training — Robert Kim, CISO

### 5.1 Security Breach Overview

Robert provided the sales team with facts to address customer concerns:

**What happened:**
- November 2025: APT41 breach discovered
- Service account vulnerability exploited
- 2.8TB data potentially accessed
- NEXL-X4 design files included

**What we did:**
- Contained within 72 hours
- Cooperated with FBI/CISA
- Customer notifications within 14 days
- $25M security transformation implemented

**Current status:**
- 85% confidence attacker removed
- Zero Trust Architecture 65% complete
- External penetration test May 2026
- Ongoing monitoring

### 5.2 Customer-Facing Messaging

**DO say:**
- "We discovered a breach and responded transparently"
- "We cooperated with authorities and notified affected customers"
- "We've invested $25M in security improvements"
- "We're confident in our security posture"
- "We're happy to discuss our security practices with your team"

**DON'T say:**
- "We were hacked by Chinese state-sponsored actors"
- "The attackers took our NEXL-X4 design"
- "We don't know if your data was affected"
- Any attribution details
- Any technical specifics not in public filings

### 5.3 Customer Response Tracking

| Customer Category | Response | Action |
|-------------------|----------|--------|
| Microsoft | Concerned, renewed anyway | Security briefing completed |
| Amazon | Evaluating, security review | Quarterly update |
| Google | Interested in X4 | Technical deep-dive scheduled |
| BMW | Supportive, timeline unchanged | Joint security review |
| 3 Enterprise Accounts | Lost | Security concerns cited |

**Key insight:** Most customers understand breaches happen. They care more about response than the breach itself.

## 6. Competitive Response — AMD MI350

### 6.1 AMD Competitive Position

James presented competitive intelligence on AMD MI350:

**AMD MI350 Strengths:**
- Competitive FP8 performance (1,850 TOPS)
- 192 GB HBM3 memory
- Aggressive pricing (~$35K)
- ROCm 6.1 improving

**AMD MI350 Weaknesses:**
- Software ecosystem still behind CUDA
- Enterprise trust building
- Limited design wins vs NVIDIA
- ROCm compatibility issues

### 6.2 Our Response

| Competitive Situation | Response |
|----------------------|----------|
| AMD undercuts on price | Emphasize performance/TOPS |
| AMD superior memory | Emphasize inference optimization |
| AMD ROCm claims | Acknowledge improvement, compare workloads |
| AMD customer claim | Reference NEXL win rate 58% vs AMD |

**Winning against AMD:**
- Focus on inference-specific workloads
- Emphasize total value (not just ASP)
- Highlight software ecosystem
- Leverage lead time advantage (18 weeks vs AMD 20+ weeks)
